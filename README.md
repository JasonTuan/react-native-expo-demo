REACT NATIVE WITH EXPO
====

# Start

## Create Project
```shell
npm install -g pnpm@latest-10
pnpm add -g expo-cli
npx create-expo-app@latest ExpoDemo
rm -rf node_modules
rm -f package-lock.json
pnpm expo install
```

## Run 
```shell
pnpm start
```
